部署與設定提醒

  1. 請先在 Google Cloud 建立 Web 應用的 OAuth Client ID 與 API Key，並把 GitHub Pages 網域加入 Authorized JavaScript origins。
  2. 若要多人共用同一頁與同一批資源，請把所有人加入相同的 Google Drive 共用資料夾或 Shared Drive，並填入相同的資料夾 ID。
  3. 如需刪除檔案，登入者需具備對該資料夾 / 檔案的對應權限。
